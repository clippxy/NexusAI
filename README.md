# ⚡ NexusAI — AI Automation Suite

> Write, summarize, generate, and organize — powered by frontier AI. Built for people who move fast.

## What is NexusAI?

NexusAI is a web-based AI automation suite that gives users access to a set of powerful AI tools — from summarizing text and generating titles, to drafting emails and building full content calendars. Powered by Claude AI.

## Features

- **Free tier** — Summarizer, Tag Suggester, Title Generator
- **Pro ($9/mo)** — Content Writer, Email Drafter, Meeting → Actions, Semantic Search
- **Elite ($29/mo)** — AI Project Planner, Content Calendar, Custom AI Agent, and everything in Pro

## Stack

- Vanilla HTML / CSS / JavaScript
- Claude AI (Anthropic) via API
- Hosted on GitHub Pages
